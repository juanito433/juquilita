  • BENVENIDDO A JUQUILITA

    JUQUILITA es un proyecto desarrollado con el fin de brindar ayuda con un sistema de gestion de inventario y venta para la fruteria JUQUILITA ubicada en Teapa, Tabasco

    Este proyecto se diseño como una aplicación web para poder hacer un fácil uso de esta para que que los usuarios/Empleados de la fruteria puedan utilizaralas sin mayor problema con una interfaz intuitiva.

    Al igual se implemento una interfaz para que clientes de manera online puedan comprar sin problemas, esto les ahorrara la molestia de ir de manera fisica a la fruteria y con ello tambien llevbara a la fruteria a un nuevo nievel en sus ventas
